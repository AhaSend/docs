<Update label="September 2025">
- Add Dark Mode to the dashboard
- Add [Service Status Page](https://status.ahasend.com)
- Add [Get Message](/api-reference/messages/get-message) API
- New marketing website
- Improve `Content-Type` and `Content-Disposition` handling for attachments in [Create Message API](/api-reference/messages/create-message)
- Add optional global suppression for large volume senders
- Fix issues with false positives in DMARC verification in some corner cases
- Fix DNS verification failures due to large TXT records: Retry truncated TXT lookups over TCP
- Improve and optimize automatic suppression system based on bounce classifications.
- Improve handling of ICS attachments in [Create Message API](/api-reference/messages/create-message)
- Add rate limits for failed login attempts.
- Improve automatic monitoring and suspension system to prevent abuse and false positives.
- Add new endpoint for sending [Conversation Messages](/api-reference/messages/create-conversation)
</Update>

- Introduce [AhaSend CLI](https://github.com/AhaSend/ahasend-cli)
- Update [documentation site](https://ahasend.com/docs)
</Update>
